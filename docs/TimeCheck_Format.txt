Интерфейс программы TimeCheck.exe ( следуйте ему если захотите написать свой для Sandra) :

Параметры программы :
1) Имя запускаемой программы (.ехе)
2) Тайм лимит для программы ( в милисекундах )

Выход :
1) одно из трех слов (без кавычек)
'okay' - если программа в отведенное время корректно завершила работу
'timelimit' - если программа не уложилась в отведенное время
'runtime' - аварийное завершение программы
2) Время выполнения в милисекундах ( если первый параметр не 'okay', то это число значения не имеет ).

Упущения :
1) Сейчас эта программа реализована на C# и гарантированно работает только в ОС Windows (хотя Ubuntu 
тоже смог запустить, но возможно для это требуется установленный Mono).
2) Она не проверяет попытки взлома системы и является самым слабым местом в безопасности. Желательно 
доработать проверки типа максимального ограничения в правах запускаемого кода.
3) Запускаемый процес никак не ограничивается кроме как временем исполнения (то есть мемори-лимит всегда 4гб).
